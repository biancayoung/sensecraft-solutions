## 这个方案能帮你做什么

在偏远牧场或离网围栏管理羊群时，无法实时掌握出入情况。这个方案将 reCamera 变成一台自动绵羊计数器，通过 LoRa 将 进/出/在栏 数量广播到您的手机和 Home Assistant 看板——即使距离数公里、围栏处没有 WiFi，也能正常工作。

## 核心价值

| 好处 | 具体说明 |
|------|---------|
| 适用于任何地点 | LoRa 无线电在视距内可传输 5 公里以上——无需在围栏处架设蜂窝网络或 WiFi |
| 全自动计数 | YOLO 模型自动计数每只穿越围栏线的绵羊，无需人工干预 |
| 手机实时告警 | 每次穿越都会立即向您的手机发送 Meshtastic 通知 |
| 持久化看板 | Home Assistant 保存实时总计和 24 小时历史，随时可查 |
| 离网可靠运行 | 每 15 分钟一次的 LoRa 心跳，在信号中断后自动重新同步计数 |

## 适用场景

| 场景 | 使用方式 |
|------|---------|
| 日常围栏点数 | 无需手动清点，直接知道有多少只羊进入或离开围栏 |
| 夜间安全监控 | 在非工作时间内，每当羊只穿越围栏即发出告警 |
| 远程牧场监控 | 农场主在城镇查看 HA 看板——无需驱车前往 |
| 多围栏追踪 | 每个围栏添加一组 reCamera + LoRa 节点，统一汇报到同一 HA 实例 |

## 使用须知

### 核心硬件（必需）

| 设备 | 说明 | 必需 |
|------|------|------|
| Seeed reCamera 2002 HQ PoE | 运行 YOLO 绵羊检测器和计数程序；RISC-V SG2002 NPU，无 WiFi | ✓ 必选 |
| XIAO ESP32-S3 + Wio-SX1262 | Meshtastic LoRa 节点——通过 UART 与 reCamera 连接，传输计数数据 | ✓ 必选 |
| reComputer R1100 或树莓派 | 网关主机——需要局域网访问权限；运行桥接服务，也可选择安装 Home Assistant | ✓ 必选 |
| Meshtastic LoRa 接收器 | 网关处的第二个 Meshtastic 设备（USB 串口或独立无线电） | ✓ 必选 |
| Home Assistant 服务器 | 显示看板并存储计数历史 | ✓ 必选 |

### 网络要求

- 部署期间，reCamera 只需通过 USB 或以太网直接连接到笔记本电脑——围栏处无需互联网
- 网关电脑需要局域网访问 Home Assistant 服务器
- LoRa 范围：Meshtastic 在默认设置下，视距内最远约 5 公里；山地或林地地形会缩短范围
